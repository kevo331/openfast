Appendix D. List of Output Channels¶

This is a list of all possible output parameters for the SubDyn module. The names are grouped by meaning, but can be ordered in the OUTPUT CHANNELS section of the SubDyn input file as the user sees fit. \(M \alpha N \beta\), refers to node \(\beta\) of member \(\alpha\), where \(\alpha\) is a number in the range [1,9] and corresponds to row \(\alpha\) in the MEMBER OUTPUT LIST table (see Section ) and \(\beta\) is a number in the range [1,9] and corresponds to node \(\beta\) in the NodeCnt list of that table entry.

Some outputs are in the SS reference coordinate system (global inertial-frame coordinate system), and end with the suffix ss; others refer to the local (member) reference system and they have suffixes “Xe”, “Ye”, or “Ze” (see Section 7).
